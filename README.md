<h1 align="center">
  Kelenken keyboard
</h1>

> **The Kelenken** is the first and only keyboard modeled entirely with **JavaScript** using the [scad-js](https://github.com/20lives/scad-js) library. it is inspired by the Dactyl keyboard and it's many descendants.
